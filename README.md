# Базовая система по поиску.

## Описание проекта
Разработана система, которая может искать и выдавать наиболее подходящие резюме по заданной вакансии, из базы данных в Telegram-бот.
Так же, можно дальше улучшать и развивать его функционал, добавляя новые функции и обрабатывая дополнительные состояния..

###  Создан через Python-код в Google Colab, используется библиотека FAISS.
 Google Colab - облачная платформа для разработки и выполнения Python-кода в браузере.
